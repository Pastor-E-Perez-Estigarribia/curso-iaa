Clase 2: Preparación de Datos

La regla de oro del Machine Learning

"Garbage in, garbage out" (Si entra basura, sale basura). El algoritmo más avanzado del mundo no puede extraer valor de datos desordenados. El preprocesamiento y la ingeniería de atributos suelen consumir hasta el 80% del tiempo de un científico de datos.

⚠️ Datos Crudos (Realidad)

  • ✗ Incompletos: Celdas vacías o valores nulos (NaN).
  • ✗ Ruidosos: Errores de medición, outliers extremos.
  • ✗ Inconsistentes: Texto mezclado con números.
  • ✗ Magnitudes Dispares: Edad (0-100) vs Salario (millones).

✨ Datos Preparados (Ideal)

  • ✓ Completos: Valores faltantes imputados estadísticamente.
  • ✓ Limpios: Outliers acotados o tratados.
  • ✓ Numéricos: Categorías codificadas matemáticamente.
  • ✓ Estandarizados: Todas las variables en escalas comparables.

Transformaciones Básicas

Interactúa con el gráfico inferior para entender por qué la Normalización es crucial. Algoritmos basados en distancias (como KNN o SVM) se confunden si una variable tiene números gigantes y otra números pequeños.

🧹 1. Limpieza e Imputación

Los algoritmos matemáticos fallan ante valores nulos. Rellenamos los vacíos con la media, mediana o moda de la columna correspondiente para no perder la fila entera.

🔤 2. Codificación (Encoding)

Convertimos texto a números. Label Encoding asigna (0,1,2...) a jerarquías. One-Hot Encoding crea columnas binarias (1 o 0) para categorías sin orden (ej. Colores).

📏 3. Normalización

Comparando variables con magnitudes dispares.

Modo Crudo: El "Salario" domina el eje Y con valores de hasta 10,000, mientras que la "Edad" en el eje X (0-10) apenas es visible para el algoritmo en el cálculo de distancias.

Selección de Atributos (Features)

Más datos no siempre es mejor. Variables redundantes ("ruido") confunden al modelo, aumentan el riesgo de sobreajuste y ralentizan el cómputo. Existen 3 metodologías para filtrar atributos.

🧮

Métodos de Filtrado

Evalúan métricas estadísticas de cada variable de forma independiente al modelo.

  • • Varianza (ANOVA)
  • • Correlación de Pearson
  • • Test de Chi-cuadrado

  • • Eliminación hacia atrás

Mapa Mental: Scikit-Learn

Las "Transformaciones del conjunto de datos" se implementan fácilmente usando módulos específicos de Scikit-Learn. Conocer estos sub-módulos te guiará al programar.

🛠️

sklearn.preprocessing

  • StandardScaler()

    Estandariza removiendo la media y escalando a varianza 1.

  • MinMaxScaler()

    Transforma características escalándolas a un rango dado (ej. 0 a 1).

  • OneHotEncoder()

    Codifica categorías en matrices numéricas binarias (dummy variables).

🎯

sklearn.feature_selection

  • SelectKBest()

    Filtro: Elimina todo menos las 'k' características con puntuación más alta.

  • RFE()

    Wrapper: Eliminación Recursiva de Características dada una estimación de pesos.
