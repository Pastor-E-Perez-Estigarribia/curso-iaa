Bienvenido al repositorio oficial del módulo **Introducción al Aprendizaje Automático**, desarrollado para como programa de Postgrado.

Este curso busca proporcionar a los estudiantes una inmersión profunda, combinando conceptos teóricos fundamentales de Machine Learning clásico y Analítica de Datos Complejos con el desarrollo de habilidades de programación aplicada.

---

## 🌐 Plataforma Interactiva del Curso

El material teórico de este módulo está diseñado en formato de **Single Page Applications (SPAs) interactivas**. 
Puedes acceder al índice principal y navegar por las clases desde cualquier dispositivo ingresando a nuestro sitio web:

👉 **[Acceder a la Plataforma Web del Módulo](https://pastor-e-perez-estigarribia.github.io/curso-iaa/)**

---

## 🎯 Objetivos del Módulo
- Examinar y comprender los conceptos fundamentales del Machine Learning clásico y profundo.
- Diferenciar y aplicar estrategias de aprendizaje supervisado, no supervisado y de analítica compleja (ej. Series Temporales, Redes Neuronales).
- Ejecutar el flujo de trabajo completo: limpieza de datos, ingeniería de atributos, entrenamiento, evaluación y ajuste de modelos.
- Desarrollar habilidades prácticas utilizando `scikit-learn` y otras librerías avanzadas en entornos de `Google Colab`.
