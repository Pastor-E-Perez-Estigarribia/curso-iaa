Aprendizaje No Supervisado

A diferencia de las clases anteriores, aquí no tenemos un objetivo (variable Y) que predecir. Solo tenemos características (X). El propósito del algoritmo es descubrir la estructura subyacente, patrones ocultos o agrupaciones naturales en los datos por sí mismo.

Supervisado (Repaso)

El profesor da ejemplos y respuestas. El alumno aprende a imitar.

X = [Edad, Salario] ➔ Y = [Compra: SÍ/NO]

No Supervisado (Hoy)

Se entregan los datos crudos. El algoritmo debe encontrar similitudes sin ayuda.

X = [Edad, Salario] ➔ [Grupo 1, Grupo 2, Grupo 3]

Clustering (Agrupamiento)

El objetivo es agrupar elementos similares. La similitud se suele medir como "distancia matemática" entre los puntos. Es vital escalar los datos (Clase 2) antes de aplicar estos algoritmos.

🎯 Simulador K-Means

Datos crudos sin etiquetas. ¿Puedes ver los grupos naturales?

1. K-Means (No Jerárquico)

Busca agrupar datos alrededor de 'K' centroides. Requiere definir 'K' de antemano.

  • ✔️ Muy rápido, escalable.
  • ❌ Asume clusters esféricos. Sensible a outliers.

2. Jerárquico

Crea un árbol de clusters (Dendrograma). Útil si no sabes cuántos grupos existen.

  • ✔️ No requiere 'K' inicial. Muy explicable.
  • ❌ Computacionalmente pesado. Lento.

3. DBSCAN

Basado en densidad. Agrupa zonas con muchos puntos juntos y aísla los solitarios.

  • ✔️ Descubre formas extrañas. Detecta ruido/outliers.
  • ❌ Falla si los clusters tienen densidades muy distintas.

PCA (Análisis de Componentes Principales)

Es la técnica de Reducción de Dimensionalidad por excelencia. Si un dataset tiene 50 variables (columnas), es imposible graficarlo. PCA comprime esa información en 2 o 3 "Componentes" principales reteniendo la mayor cantidad de información (varianza) posible.

¿Para qué sirve PCA?

  • 👀 Visualización: Bajar de N dimensiones a 2D para ver los clusters en un gráfico.
  • ⚡ Acelerar Modelos: Entrenar modelos sobre 5 componentes comprimidos en vez de 100 variables originales.
  • 🧹 Eliminar Ruido: Ignorar variables altamente correlacionadas o irrelevantes.

Pérdida de interpretabilidad a cambio de compresión.

Detección de Anomalías

Otra gran aplicación del No Supervisado. Consiste en identificar instancias que se desvían significativamente del comportamiento normal ("outliers").

💳

Fraude Bancario

Identificar transacciones de tarjetas de crédito que no encajan en el patrón habitual del cliente.

⚙️

Mantenimiento Predictivo

Detectar vibraciones o temperaturas anómalas en maquinaria industrial antes de que falle.

Reto Colab: PCA + K-Means

Integraremos dos conceptos. Usaremos un dataset real multidimensional, reduciremos sus dimensiones a 2 con PCA para poder graficarlo, y luego le aplicaremos K-Means para ver cómo agrupa la información.

"En Python con scikit-learn, carga el dataset 'load_wine' (13 dimensiones).

2. Usa PCA para reducir los datos a 2 componentes principales (PCA(n_components=2)).
3. Aplica KMeans(n_clusters=3) a los datos reducidos por PCA.
4. Genera un gráfico de dispersión (scatter plot) donde el eje X sea el PC1, el eje Y sea el PC2, y el color de los puntos represente las etiquetas predichas por K-Means."
