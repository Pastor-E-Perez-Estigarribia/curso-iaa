El límite de la linealidad

En la clase anterior vimos modelos simples. Sin embargo, en el mundo real, los datos rara vez se pueden separar trazando una simple "línea recta" (como hace la Regresión Logística). Cuando los datos tienen relaciones complejas y no lineales, necesitamos modelos con mayor "fuerza bruta" matemática.

La limitación Clásica

Imagina un anillo de puntos azules rodeado por un anillo de puntos rojos. Ninguna línea recta trazada de lado a lado podrá separarlos correctamente.

Regresión Logística -> ❌ Falla

La Solución Avanzada

Modelos como SVM utilizan el "Truco del Kernel" para doblar el espacio, y las Redes Neuronales utilizan múltiples capas para "aprender" curvas y fronteras irregulares.

SVM / Redes Neuronales -> ✅ Éxito

Máquinas de Vectores de Soporte (SVM)

A diferencia de otros modelos que solo buscan separar las clases, SVM busca la "calle más ancha" (el hiperplano con el mayor margen) entre las dos clases. Los puntos de datos que tocan los bordes de esta "calle" se llaman Vectores de Soporte.

El "Truco del Kernel" (Kernel Trick)

Si los datos no son separables con una línea en 2D, SVM utiliza funciones matemáticas (Kernels) para proyectar los datos a una tercera dimensión temporalmente, donde sí se pueden separar con un plano plano, y luego los devuelve al 2D revelando una curva.

  • 🔵 Linear: Para datos simples (línea recta).
  • 🟢 Poly: Fronteras curvas polinómicas.
  • 🔴 RBF (Radial Basis Function): Fronteras circulares o islas aisladas (el más usado).
🛣️

Robustez

Al basarse solo en los vectores de soporte (los puntos límite), SVM es muy resistente a valores atípicos lejanos.

Perceptrón Multicapa (MLP)

Es la arquitectura fundamental de las Redes Neuronales Artificiales. Inspirado en el cerebro humano, utiliza capas de "neuronas" interconectadas. Cada neurona recibe señales, aplica una función de activación matemática, y pasa el resultado a la siguiente capa.

En Scikit-Learn lo implementamos con MLPClassifier. A diferencia del Deep Learning moderno (TensorFlow/PyTorch), MLPClassifier es ideal para conjuntos de datos pequeños o medianos en formato de tabla (tabular), pero no para procesar imágenes masivas o texto profundo.
